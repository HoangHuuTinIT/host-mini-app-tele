# Đây là 1 repo chứa app android đã tích hợp 1 mini-app dạng web

## Clone Repository

```bash
git clone https://github.com/HoangHuuTinIT/host-mini-app-tele.git
```

---

## Chạy App
### Sau khi clone repo về, thực hiện các bước sau
1. Mở **Android Studio**
2. Chọn **File → Open**
3. Tìm đến nơi đặt repo vừa clone về
4. Mở repo trong android studio 
5. Đợi Gradle sync hoàn tất
6. Chạy app
